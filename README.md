# 🔍 Debounced Search App

An optimized search application built using **JavaScript debounce technique** to reduce unnecessary API calls and improve frontend performance.

---

## 🚀 Features

* 🔹 Debounced input handling (prevents excessive API calls)
* 🔹 Real-time search results
* 🔹 Clean and minimal UI
* 🔹 API integration (GitHub / DummyJSON)
* 🔹 Performance optimization using event control

---

## 🧠 Concepts Used

* Debouncing
* Event Handling
* API Fetching
* DOM Manipulation

---

## 🛠️ Tech Stack

* HTML
* JavaScript (Vanilla JS)

---

## ⚡ How It Works

* User types in the search box
* Debounce delays the API call until typing stops
* Only the final input triggers the API request
* Results are displayed dynamically
